  - Release approvals
- Security requirements
  - Data encryption
  - Data transmission
  - De-identify data/data masking
- Storage environment requirements
  - Shared drive vs. cloud based vs. local storage
- Use requirements
  - Acceptable use policy
  - Data processing
  - Data deletion
  - Data retention

  - Record link restrictions
  - Data constraints
  - Cardinality
